PalmのGraffiti風ストローク入力キーボードを追加。

* 日英切り替えは「左下→右上」のストローク
* 絵文字等のシンボル入力は「左上→右下」のストローク
* オプションメニュー表示は「左下→右上→左下」のストローク

ビルド済みバイナリはこちら→[apkファイル](./MozcForAndroid.apk)
